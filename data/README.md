# Input data

Large input datasets are not included in this GitHub repository.

Expected input files include:

- paired-end metagenomic reads from NCBI SRA;
- MAG FASTA files from the reconstructed/dereplicated MAG catalogue;
- MAG taxonomy and quality tables;
- ARG and VFDB annotation tables;
- MAG abundance matrices;
- sample metadata and environmental predictors;
- phylogenetic tree files;
- derived matrices deposited in the public data repository described in the manuscript.

Users should edit the templates in `config/` so that the scripts point to their local copies of these files.
