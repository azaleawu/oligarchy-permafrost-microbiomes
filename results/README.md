# Results directory

This directory is reserved for generated workflow outputs. Large intermediate files are not stored in this repository.

Expected outputs include:

- assembly and mapping summaries;
- MAG binning, dereplication, quality and taxonomy outputs;
- ARG and VFDB annotation summaries;
- MAG abundance matrices;
- analysis-ready matrices;
- statistical summary tables;
- source-data tables;
- analytical plotting panels.
