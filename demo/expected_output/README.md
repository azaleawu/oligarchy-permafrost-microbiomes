# Expected demo output

Running

```bash
Rscript demo/run_demo.R
```

will create:

- `results/demo_outputs/demo_sample_burdens.tsv`
- `results/demo_outputs/demo_model_summary.tsv`
- `results/demo_outputs/demo_readme.txt`

The exact numerical values are determined by the simulated demo data in `demo/` and are not intended to match manuscript results.
