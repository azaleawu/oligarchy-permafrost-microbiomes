Software versions used in the manuscript workflow

Read trimming: Trimmomatic v0.39
Metagenome assembly: MEGAHIT v1.2.9
Read mapping: Bowtie2 v2.5.4
BAM processing: Samtools v1.x (record exact local version when rerunning)
MAG binning: VAMB v4.1.3
MAG dereplication: dRep v3.4.2
Genome quality assessment: CheckM2 v1.0.2
Taxonomic assignment: GTDB-Tk v2.4.0 with GTDB release R226
Abundance profiling: CoverM v0.7.0
ORF prediction: Prodigal v2.6.3
ARG annotation: RGI v6.0.2 with CARD v3.2.9
Virulence-associated gene annotation: ABRicate v1.0.1 with VFDB
Statistical analysis: R 4.x
