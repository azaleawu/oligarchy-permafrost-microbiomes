# Code and software checklist notes

This repository is structured to support the Nature Research Code and Software Submission Checklist.

## Required content

- Source code is provided in `scripts/`.
- A small simulated demo dataset is provided in `demo/`.

## README content

The main `README.md` includes:

- system requirements
- software dependencies
- operating system expectations
- non-standard hardware requirements for full-scale analyses
- installation guide
- expected installation time
- demo instructions
- expected demo output
- expected demo runtime
- instructions for use on real data
- reproduction notes
