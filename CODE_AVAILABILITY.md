# Code availability

This repository contains cleaned workflow scripts and downstream analysis code supporting the manuscript:

**An oligarchy of microbes and genes control the fate of ARGs and pathogens in permafrost soils worldwide**

The repository includes code for:

- metagenome assembly, mapping and MAG reconstruction
- MAG quality assessment, dereplication and taxonomic assignment
- ORF prediction
- ARG annotation using RGI/CARD
- virulence-associated gene annotation using ABRicate/VFDB
- MAG abundance profiling
- matrix construction
- phylogenetic diversity and structure analyses
- environmental driver screening
- multivariable mixed-effect GAMM analyses
- phylogenetic factorisation and hotspot analyses
- hotspot overlap and enrichment analyses
- analytical-panel and source-data export

Large input data and large intermediate files are not included. These include raw FASTQ files, BAM files, assemblies, external databases and full software output directories. These data are available from NCBI SRA, GenBank, Zenodo and Source Data files as described in the manuscript.

A versioned release of this repository will be archived in a DOI-minting repository, such as Zenodo, upon publication.
